Google Cloud Java Client -- Core
=========================================

This module provides common functionality required by service-specific modules of this library.

Quickstart
----------
If you are using Maven, add this to your pom.xml file
```xml
<dependency>
  <groupId>com.google.gcloud</groupId>
  <artifactId>gcloud-java-core</artifactId>
  <version>0.0.12</version>
</dependency>
```
If you are using Gradle, add this to your dependencies
```Groovy
compile 'com.google.gcloud:gcloud-java-core:jar:0.0.12'
```
If you are using SBT, add this to your dependencies
```Scala
libraryDependencies += "com.google.gcloud" % "gcloud-java-core" % "0.0.12"
```

Java Versions
-------------

Java 7 or above is required for using this client.
